Add Date Sort Order Column Migration

Supabase Configuration

What this migration does:

  • Adds a 'date_sort_order' column to the 'column_settings' table
  • Sets default value to 'descending'
  • Adds a check constraint to ensure only 'ascending' or 'descending' values
  • Updates existing records with the default value
ALTER TABLE column_settings ADD COLUMN date_sort_order VARCHAR(10) DEFAULT 'descending'; ALTER TABLE column_settings ADD CONSTRAINT check_date_sort_order CHECK (date_sort_order IN ('ascending', 'descending')); UPDATE column_settings SET date_sort_order = 'descending' WHERE date_sort_order IS NULL;
